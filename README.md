# E-commerce Sales Dashboard | Power BI

## Overview
This project presents an interactive Power BI dashboard designed to analyze e-commerce sales performance across multiple dimensions such as sales amount, quantity, profit, payment modes, categories, states, and time periods. The dashboard enables quick monitoring of key business metrics and trends.

## Key KPIs
- **Total Amount:** 438K  
- **Total Quantity Sold:** 5,615  
- **Total Profit:** 37K  
- **Overall Transactions:** 2M  

## Dashboard Analysis
The dashboard provides insights into:
- State-wise sales distribution  
- Monthly sales and profit trends  
- Category-wise and sub-category-wise profit analysis  
- Payment mode usage (COD, UPI, Debit Card, Credit Card, EMI)  
- Customer-wise order distribution  

## Visuals Included
- KPI cards for high-level performance tracking  
- Bar charts for state-wise and customer-wise analysis  
- Line/column chart for monthly profit trends  
- Donut charts for category and payment mode distribution  
- Horizontal bar chart for sub-category profit comparison  

## Tools & Technologies
- Power BI  
- DAX  
- Excel / CSV Dataset  
- Data Cleaning & Data Modeling
